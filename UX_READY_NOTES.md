# TripleSide Studio — UX-ready license flow

## Customer flow
1. Open product.
2. Choose **Buy Now** or **Try Free**.
3. The serial is generated automatically, emailed, and shown in **My Licenses**.
4. The plugin only asks for the serial; hardware identification is automatic.

## Admin product flow
License settings are intentionally short:
- Protect this plugin with a license
- Computers allowed: 1–3
- Offer a free trial
- Trial length

Fields that are not relevant remain hidden.

## Device management
Customers can copy their serial and deactivate a computer from the dashboard without contacting support.
